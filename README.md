# RealtyHub Pro - Gelişmiş Emlak Yönetim Sistemi

RealtyHub Pro, modern emlak sektörünün tüm ihtiyaçlarını karşılayan, gelişmiş özelliklere sahip bir emlak yönetim sistemidir. Laravel 11 ile geliştirilmiş bu platform, emlak profesyonellerine ve emlak şirketlerine özel çözümler sunar.

## 📦 Kurulum

```bash
# Projeyi klonlayın
git clone https://github.com/your-username/realtyhub-pro.git

# Bağımlılıkları yükleyin
composer install
npm install

# Ortam dosyasını oluşturun
cp .env.example .env

# Uygulama anahtarını oluşturun
php artisan key:generate

# Veritabanını migrate edin
php artisan migrate --seed

# Frontend varlıklarını derleyin
npm run build
```
